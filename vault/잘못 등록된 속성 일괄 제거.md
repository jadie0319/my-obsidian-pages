# 잘못 등록된 속성 일괄 제거

```
grep -RIn --include="*.md" -E '^(date created|date modified|created|modified|data created|data modified)[[:space:]]*:' .

find . -name "*.md" -type f -print0 | xargs -0 perl -i -pe 's/^\s*(date created|date modified|created|modified|data created|data modified)\s*:\s*.*\n//gmi'
```
